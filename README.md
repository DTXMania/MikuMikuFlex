# MikuMikuFlex

MMF (Miku Miku Flex Library) は、.NET Framework で利用可能な MMD (Miku Miku Dance) ライブラリです。

MMF公式の開発が終了してしまったようなので、それを独自に改変したものです。

## 動作環境

### 動作環境
* Windows 10 (x86/x64)
* .NET Framework 4.7.1+
* DirectX 11.0+

### 開発環境
* Windows 10 Pro (x64)
* Visual Studio 2017 Community
